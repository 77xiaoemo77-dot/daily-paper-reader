## 摘要
中长期股票配置因预测结构较弱、市场机制非平稳、以及在考虑交易成本、容量限制和尾部风险约束后信号退化等问题而面临显著挑战。传统方法通常依赖单一预测模型或松耦合的“预测—配置”流程，因此在分布漂移情形下鲁棒性不足。本研究探讨了一个有针对性的问题：将强化学习（RL）、多智能体策略群体、基于策略空间响应Oracle（PSRO）的聚合、联赛式最优响应训练、进化式替换机制，以及考虑执行因素的检查点选择，整合到一个统一的滚动回测闭环中，是否能够提升中长期资产配置的鲁棒性。所提出的框架EvoNash-MARL将上述模块整合入一个考虑执行因素的配置循环中，并进一步引入分层策略结构（包括方向头和风险头）、非线性信号增强、特征质量重加权以及约束敏感的检查点选择。在120期滚动窗口的实验协议下，最终的v21配置实现了平均超额Sharpe比0.7600与稳健得分-0.0203，内部对照组中排名第一；在2014-01-02至2024-01-05的日度样本外收益上，获得19.6%的年化收益（相比SPY的11.7%），并在延伸至2026-02-10的滚动评估中实现20.5%（相比13.5%）。该框架在现实压力约束下仍保持正向表现，并表现出结构化的跨市场泛化能力；然而，在White现实检验（WRC）及SPA-lite测试下，并未确立全局显著优势。因此，结果被视为支持一种更稳定的中长期训练与选择范式的证据，而非普遍优越的市场择时性能证明。
