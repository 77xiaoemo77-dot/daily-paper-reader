## 摘要
机器学习模型可以通过学习数据样本来高效地执行各种任务。当数据样本遭到对抗性操控，例如加入精心构造的噪声时，模型可能会出现错误。量子机器学习模型同样易受到此类对抗性攻击的影响，特别是在使用变分量子分类器进行图像分类时。尽管已有一些针对这些对抗扰动的防御方法（如使用对抗样本进行训练）展现出潜力，但这些方法存在实际局限性。例如，在无法使用对抗样本进行训练的场景中，或模型可能因某一类攻击而过拟合，这些方法就不适用。本文提出一种基于量子自编码器的免对抗训练防御框架，通过重构过程净化对抗样本。此外，我们的防御框架提供了一种置信度度量，用以识别无法被量子自编码器净化的潜在对抗样本。广泛的实验证明，在遭受对抗攻击时，该防御框架在预测准确率方面可显著优于现有技术（最高提升达68%）。

## Abstract
Machine learning models can learn from data samples to carry out various tasks efficiently. When data samples are adversarially manipulated, such as by insertion of carefully crafted noise, it can cause the model to make mistakes. Quantum machine learning models are also vulnerable to such adversarial attacks, especially in image classification using variational quantum classifiers. While there are promising defenses against these adversarial perturbations, such as training with adversarial samples, they face practical limitations. For example, they are not applicable in scenarios where training with adversarial samples is either not possible or can overfit the models on one type of attack. In this paper, we propose an adversarial training-free defense framework that utilizes a quantum autoencoder to purify the adversarial samples through reconstruction. Moreover, our defense framework provides a confidence metric to identify potentially adversarial samples that cannot be purified the quantum autoencoder. Extensive evaluation demonstrates that our defense framework can significantly outperform state-of-the-art in prediction accuracy (up to 68%) under adversarial attacks.

## 二、方法论与技术细节

### 1. 总体框架：QAE++  
论文提出的防御框架称为 **QAE++（Quantum Autoencoder Defense Framework）**，主要由三部分组成：
1. **量子自编码器（Quantum Autoencoder, QAE）**：对输入样本进行量子态编码与重构，消除或约减对抗扰动；
2. **变分量子分类器（VQC）**：执行最终分类任务；
3. **置信度度量模块（Confidence Metric）**：判断样本是否可信或应被拒绝。

### 2. QAE 工作原理
- **编码与解码：**  
  将输入数据嵌入到 n 个量子比特，通过参数化量子线路 **U(θ)** 执行压缩（encoder）和逆操作（decoder）。
- **压缩目标：**  
  将输入态 |ψ_in⟩ 变换为低维潜在空间态 |ψ_A⟩，并使“废弃空间”态 |ψ_B⟩ 与参考态 |0⟩⊗(n−k) 具有最大保真度。
- **训练机制：**  
  仅需训练编码器，通过 **SWAP Test** 测量保真度 ⟨σ_Z⟩，损失函数为：  
  \[
  L = 1 - \langle σ_Z \rangle
  \]
  最大化保真度即最小化 L。

### 3. 置信度度量计算
防御策略通过量化样本可信度来判定是否接受模型预测。  
- **组成部分：**
  - **编码保真度（Encoding Fidelity, ⟨σZ⟩ₓ）**
  - **分类器输出差值（Logit Difference, l̂ₓ）**
- **线性组合公式：**
  \[
  C = \langle σ_Z \rangle_x + \frac{l̂_x}{2}
  \]
- **阈值计算机制：**
  基于验证集构建置信度阈值 T，并在推理阶段判定：
  如果 \( C < T \)，则拒绝样本 (判定为潜在对抗样本)；否则接受分类预测。

### 4. 算法流程（QAE++）
1. 输入样本 x → QAE → 重构样本 \(\hat{x}\)、保真度 ⟨σZ⟩ₓ；
2. 重构样本输入 VQC → 得到各类别 logit 向量 z；
3. 计算 logit 差值 l̂ₓ；
4. 综合计算置信度 C 并与阈值 T 比较；
5. 若低于阈值则拒绝样本，否则输出预测标签。

## 三、实验设计

### 1. 数据集与场景
- **MNIST**：手写数字（28×28灰度图，调整为32×32以适配10量子比特的振幅嵌入）。  
- **Fashion-MNIST (FMNIST)**：服饰图像数据集，结构与MNIST类似。  
- 每个数据集包含：
  - 60,000训练样本  
  - 10,000测试样本  
  - 其中2,000样本作为验证集，用于确定置信阈值。

### 2. 攻击与防御 Benchmark
- **攻击类型：**
  - Fast Gradient Sign Method (FGSM)  
  - Projected Gradient Descent (PGD)
- **扰动强度（ϵ）：** 0.05 至 0.30
- **防御比较对象：**
  1. 无防御（None）  
  2. 基于对抗训练的经典自编码器（Classical Autoencoder, CAE）  
  3. 单纯使用 QAE 重构  
  4. 完整框架 QAE++（含置信度判断）  

### 3. 模型配置
- 评估三种不同层数的 VQC：100层、200层、300层。
- 准确率提升最高达 **68%**（在 MNIST FGSM ϵ=0.30 下，QAE++相较CAEs）。
