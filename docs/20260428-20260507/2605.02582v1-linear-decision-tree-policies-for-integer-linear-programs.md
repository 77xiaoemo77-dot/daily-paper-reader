## 摘要
我们研究了具有固定可行集合和变化成本向量的整数线性规划的最优决策策略，这些策略被表示为线性决策树。一旦针对给定的可行集合合成完成，它们就可以通过一系列线性测试针对任意查询的成本向量返回最优解。我们证明存在一种策略，在最坏情况下，可通过多项式数量的算术运算实现这一操作。除了这一理论保证外，我们还开发了一个实用的构建框架，用于在特定的线性决策树子类中合成策略。我们的计算实验表明，尽管策略合成可能耗时较长，但在重复查询时，它能够以比传统和专用求解方法快几个数量级的速度获取最优解。总体而言，这一范式为整数线性规划的复杂性提供了新的视角，并为其提供了一种离线--在线求解方法。

## Abstract
We study optimal decision policies for integer linear programs with a fixed feasible set and varying cost vectors, represented as linear decision trees. Once synthesized for a given feasible set, they return an optimal solution for any queried cost vector through a sequence of linear tests. We show that there exists a policy performing this operation in a polynomial number of arithmetic operations in the worst case. Along with this theoretical guarantee, we develop a practical construction framework to synthesize policies within a specific subclass of linear decision trees. Our computational experiments show that, although policy synthesis can be time-intensive, it allows retrieving optimal solutions orders of magnitude faster than classical and specialized solution methods on repeated queries. Overall, this paradigm provides a new perspective on the complexity of integer linear programs and offers an offline--online approach for solving them.

## 一、核心问题与整体含义

- **研究动机**：  
  各类整数线性规划（ILP）问题在运筹、金融、网络设计等领域广泛存在。传统算法（如分支定界、动态规划）在**每次成本向量变化时都需重新求解**，未能充分利用固定可行集合结构。  
- **研究目的**：  
  论文旨在提出一种可**离线构建、在线查询**的策略，使得在固定可行集合 \( X \subset \mathbb{Z}^n \) 下，能对任意成本向量 \( c \in C \subset \mathbb{R}^n \) 高效返回最优解。  
- **研究意义**：  
  提出一种新型“**线性决策树（Linear Decision Tree, LDT）策略**”，既能理论上保证多项式查询复杂度，又在实践中显著提升多次查询的速度，为 ILP 提供离线—在线优化的新范式。

## 二、方法论：核心思想与关键技术

### 1. 基本思想
- 将“求解 ILP”转化为“构建决策策略 π : C → X”，使每次查询仅需一系列线性比较操作（如 \( a^\top c + b \) 与 0 比较）。
- 策略形式化为线性决策树（LDT）：  
  - 每个节点执行一个线性测试（如 \( a^\top c + b \in \{<,=,>\} \)）。  
  - 树的叶节点输出最优可行解 \( x \in X \)。  
- 通过离线阶段对 \( X \) 的几何结构（如凸包与法锥扇形 Normal Fan）进行分析和编码，从而快速定位 \( c \) 属于哪一个最优区域。

### 2. 理论基础
- 引入 **Normal Fan** 概念：  
  定义每个可行点的最优性锥 \( F(x) = \{ c | c^\top x \ge c^\top \tilde{x}, \forall \tilde{x} \in X \} \)。  
  查询最优解即确定 \( c \) 属于哪个锥。  
- 证明（基于 Kolinek 1987 的结果）：  
  对任意 \( X \subset \mathbb{Z}^n \)，存在深度  
  \[
  D \approx 2 n^4 \log_2(n) + 2 n^4 \log_2(M) + O(n^3)
  \]
  的三叉 LDT，可在多项式数量算术操作下完成最优解查询。

### 3. 实际构建算法
- 提出 **动态规划与剪枝算法（Algorithm 3.1）**：  
  - 递归计算节点最小高度 \( \Delta(N) \)，选择分隔超平面（divider）使树深度最小。  
  - 利用剪枝条件 \( d(N) + \Delta_{lb}(N) > \Delta_{ub}(N_0) \) 剔除不必要节点。  
  - 引入下界和上界估计、迭代加深策略（控制检验分隔面数量）。  
- **加速机制**：  
  - 锥体与分隔推理规则，不再重复求解线性可行性问题。  
  - 按分割平衡度（variance-based discrepancy）排序分隔面以优化构造路径。  
- 最终获得一个最小深度（或近似最小）的 LDT，可离线存储、在线快速查询。

## 三、实验设计

### 1. 实验对象与场景
- 选取三个典型 ILP 问题类别：
  1. **Knp**：背包问题（Knapsack）；
  2. **Tsp**：旅行商问题（Traveling Salesman）；
  3. **Cut**：最小割问题（Min-Cut）。  
- 每类问题定义固定可行集 \( X \)，改变维度参数 \( d \) 代表问题规模。

### 2. 对比方法（Benchmark）
- **LDT 策略（提出方法）**：基于 Algorithm 3.1 构建最小深度决策树。
- **Convex-hull 策略**：构造可行集合凸包并在查询阶段以单纯形法求解线性松弛。  
- **传统优化方法**：如 Bellman、Stoer–Wagner、Held–Karp 算法。  
- **Brute-force**：穷举所有可行解。

## 六、主要结论与发现

- **理论结果**：  
  - 存在可在多项式次数算术操作内求最优解的 LDT 策略；
  - 对于二进制可行集，等价于最近邻搜索问题，可用 NNS 方法加速。
- **实践结果**：  
  - LDT 策略在查询阶段比单纯形法快几个数量级；
  - 对多数实例，LDT 查询步数 < Convex-hull 策略最佳情况；
  - NNS 构造代价低但查询仍具竞争力。  
- 总体上，**LDT 策略显著提升可重查询场景的优化效率**，验证其理论与实际可行性。
