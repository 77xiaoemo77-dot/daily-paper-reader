## 摘要
将 Answer Set Programming（ASP）与神经网络集成已成为神经符号人工智能（Neuro-symbolic AI）中的一项有前景的工具。现有方法虽将 ASP 的能力扩展到了现实世界领域，但其推理流程依赖于传统求解器，这在可扩展性上构成瓶颈。为解决这一问题，我们提出了一种新的稳定模型计算方法，称为决策传播（Decision-Propagation，DProp），该方法在否定决策与真值传播之间交替进行。研究表明，成功的 DProp 计算能够捕获稳定模型语义。随后，我们开发了一种可微分的 DProp 扩展——神经 DProp（Neural DProp，NDProp），它采用神经计算进行决策，并使用模糊评估来进行传播。我们评估了 NDProp 在学习决策启发式以及神经符号集成中的能力，并将其与现有神经符号方法进行比较。结果显示，NDProp 能够学习高效地计算稳定模型，并在神经符号基准测试中提升准确率与可扩展性。

## Abstract
Integration of Answer Set Programming (ASP) with neural networks has emerged as a promising tool in Neuro-symbolic AI. While existing approaches extend the capabilities of ASP to real world domains, their reasoning pipelines depend on classical solvers, which is a bottleneck for scalability. To tackle this problem, we propose a new method to compute stable models, called decision-propagation (DProp), which alternates falsity decisions and truth propagations. Successful DProp computations are shown to capture the stable model semantics. We then develop Neural DProp (NDProp), a differentiable extension of DProp with neural computation for decisions and fuzzy evaluation for propagations. We evaluate the capabilities of NDProp for learning decision heuristics as well as neuro-symbolic integration, and compare it with existing neuro-symbolic approaches. The results show that NDProp can learn to efficiently compute stable models, and it improves accuracy and scalability on neuro-symbolic benchmarks.

## 二、方法论：DProp 与 NDProp

### 2.1 决策传播（Decision-Propagation, DProp）
- **核心思想**  
  - 使用一种交替迭代式过程计算 ASP 稳定模型。  
  - 主要由两个步骤循环进行：
    1. **决策（Decision）阶段**：在未决原子（undecided atoms）中非确定性地选取并将其标记为“假”；
    2. **传播（Propagation）阶段**：基于已确定的假设，向前推理其他可确定为真的原子。

- **形式逻辑结构**
  - 定义包含正负体（positive/negative bodies）的规则，并通过 **迭代闭包算子 \( C_{P,J} \)** 更新真值集，直至达到稳定。
  - 理论上证明：若该过程终止且无冲突（即无同一原子既真又假），其结果恰好对应 ASP 的稳定模型。

- **算法流程特征**
  - 可在有限原子集上保证终止；
  - 决策可以批量进行；
  - 每次迭代均保持单调增加的真集（T）与假集（F）。

### 2.2 神经决策传播（Neural DProp, NDProp）
- **目标**：将 DProp 拓展为可微结构，使其能融入神经网络框架。  
- **关键设计**：
  - 使用神经网络（RNN）生成 **决策分数（决策为假的概率）**；
  - 用模糊逻辑算子（t-norm/t-conorm）实现“传播”阶段的软化逻辑；
  - 每个原子由真度 τ 与假度 ϕ 表示，其组合为软真值 \( p_i = \frac{1}{2}(\tau_i + 1 - \phi_i) \)；
  - 决策与传播均为可微计算图，允许通过梯度学习；
  - 当输出趋于二值化（binary）时，NDProp 等价于 DProp，因而可保证语义一致性。

- **理论性质**
  - 证明 NDProp 的二值极限对应稳定模型；
  - 引入损失对齐（loss alignment）定理：若强制输出为二值（使用二元交叉熵损失），模型自然学习到稳定模型语义。

## 三、实验设计与对比基线

### 3.1 实验目标
作者围绕三个问题展开实验：
1. **NDProp 能否学习 ASP 求解的决策动态？**  
2. **NDProp 在神经符号系统中能否超越传统 ASP 求解器？**  
3. **NDProp 是否能在模糊或不完备数据下进行稳健推理？**

### 3.2 数据与场景

1. **随机逻辑程序（Random Logic Programs）**
   - 使用两类生成器：负双文字程序（N2L）与 3-Literal 程序（3-LP）。
   - 涵盖 Easy (5–10 atoms), Medium (11–50), Hard (51–100) 三个难度。
   - 测试 NDProp 学习稳定模型搜索策略的能力。
   - 对比：  
     - **RDProp-k**（随机决策版本，含 1/10/100 次重启）  
     - NDProp（训练 1000 epochs）。

2. **神经符号任务（MNIST Arithmetic Benchmarks）**
   - 基于图像的推理任务：Addition、Addition2、Add2x2、Apply2x2、Member3、Member5。  
   - 只通过算术或集合约束提供监督（无单独标签）。  
   - 对比系统：
     - **NeurASP**（基于经典 ASP 求解）  
     - **SLASH**（结合概率及神经组件的系统）。

3. **不完备数据推理（Visual Sudoku）**
   - MNIST 数字作为棋盘填充内容（4×4 Sudoku）。
   - 模拟不同程度的训练数据不足（1%~100%）以检验鲁棒性。
   - 对比：  
     - **ASP（传统求解）**
     - **纯神经模型（Neural baseline）**
