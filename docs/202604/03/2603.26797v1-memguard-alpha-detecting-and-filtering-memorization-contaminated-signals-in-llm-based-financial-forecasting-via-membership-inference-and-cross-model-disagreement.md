## 摘要
大语言模型（LLMs）越来越多地被用于生成金融 Alpha 信号，但越来越多的证据表明，LLMs 会记忆其训练语料库中的历史金融数据，从而产生虚假的预测准确率，而这种准确率在样本外会发生崩溃。这种由记忆引起的先窥偏差（look-ahead bias）威胁到基于 LLM 的量化策略的有效性。先前的补救措施——模型重训练和输入匿名化——要么成本极高，要么会引入显著的信息损失。目前尚无方法能为实时交易提供实用且零成本的信号级过滤。我们推出了 MemGuard-Alpha，这是一个生成后处理框架，包含两种算法：(i) MemGuard 综合评分（MCS），它通过逻辑回归将五种成员推理攻击（MIA）方法与时间邻近特征相结合，在污染分离方面实现了 Cohen's d = 18.57（仅使用 MIA 特征时 d = 0.39-1.37）；(ii) 跨模型记忆不一致性（CMMD），它利用不同 LLM 之间训练截止日期的差异，将记忆信号与真实的推理区分开来。在对 7 个 LLM（参数量 124M-7B）、50 只标普 100 指数成份股、42,800 个提示词以及 5 种 MIA 方法进行的为期 5.5 年（2019-2024 年）的评估中，CMMD 实现了 4.11 的夏普比率，而未过滤信号的夏普比率为 2.76（提升了 49%）。清洁信号产生的平均日收益率为 14.48 bps，而受污染信号仅为 2.13 bps（相差 7 倍）。研究中出现了一个显著的交叉模式：样本内准确率随污染程度增加而上升（从 40.8% 升至 52.5%），而样本外准确率则下降（从 47% 降至 42%），这为记忆以牺牲泛化能力为代价虚增表观准确率提供了直接证据。

## Abstract
Large language models (LLMs) are increasingly used to generate financial alpha signals, yet growing evidence shows that LLMs memorize historical financial data from their training corpora, producing spurious predictive accuracy that collapses out-of-sample. This memorization-induced look-ahead bias threatens the validity of LLM-based quantitative strategies. Prior remedies -- model retraining and input anonymization -- are either prohibitively expensive or introduce significant information loss. No existing method offers practical, zero-cost signal-level filtering for real-time trading. We introduce MemGuard-Alpha, a post-generation framework comprising two algorithms: (i) the MemGuard Composite Score (MCS), which combines five membership inference attack (MIA) methods with temporal proximity features via logistic regression, achieving Cohen's d = 18.57 for contamination separation (d = 0.39-1.37 using MIA features alone); and (ii) Cross-Model Memorization Disagreement (CMMD), which exploits variation in training cutoff dates across LLMs to separate memorized signals from genuine reasoning. Evaluated across seven LLMs (124M-7B parameters), 50 S&P 100 stocks, 42,800 prompts, and five MIA methods over 5.5 years (2019-2024), CMMD achieves a Sharpe ratio of 4.11 versus 2.76 for unfiltered signals (49% improvement). Clean signals produce 14.48 bps average daily return versus 2.13 bps for tainted signals (7x difference). A striking crossover pattern emerges: in-sample accuracy rises with contamination (40.8% to 52.5%) while out-of-sample accuracy falls (47% to 42%), providing direct evidence that memorization inflates apparent accuracy at the cost of generalization.

#### 1. 核心问题与研究动机
*   **核心问题**：大语言模型（LLM）在金融预测中存在严重的“记忆效应”（Memorization）。由于其训练数据包含了历史股价和财务信息，模型在进行回测时往往是在“背诵”答案而非进行逻辑推理，导致严重的**前瞻偏差（Look-ahead Bias）**。
*   **研究动机**：现有的解决方法（如重新训练模型或数据匿名化）要么成本极高（数百万美元），要么会损失关键的金融信息。市场急需一种低成本、实时、且能在信号层面过滤受污染预测的框架，以确保量化策略在样本外（Out-of-Sample）的真实有效性。

#### 2. 方法论：核心思想与技术细节
MemGuard-Alpha 框架由两个核心算法组成：
*   **MemGuard 综合评分 (MCS)**：
    *   **核心思想**：利用成员推理攻击（MIA）技术来量化一个特定预测信号被模型“记忆”的概率。
    *   **技术细节**：整合了五种 MIA 指标：
        1.  **Loss**：平均负对数似然，值越低表示模型越熟悉。
        2.  **Min-K% Prob**：关注预测概率最低的 K% 个 Token，识别特定实体（如精确价格）的记忆。
        3.  **Min-K%++**：对 Min-K% 进行位置校准，排除 Token 本身难度的干扰。
        4.  **Zlib Ratio**：将模型 Loss 与 Zlib 压缩熵对比，区分真正的记忆与简单的文本结构。
        5.  **Reference Model**：对比目标模型与参考模型（如 GPT-2 base）的 Loss 差异。
    *   **公式逻辑**：通过逻辑回归将上述 MIA 特征与**时间邻近特征**（样本日期与模型训练截止日期的距离）结合，输出统一的污染概率。
*   **跨模型记忆不一致性 (CMMD)**：
    *   **核心思想**：利用不同 LLM 具有不同训练截止日期（Cutoff Dates）的天然特性。
    *   **流程**：对于同一个交易信号，计算多个模型的 MCS 评分。根据中位数将模型分为“清洁组”和“受污染组”。最终交易信号仅由“清洁组”的共识决定。

    *   42,800 个金融提示词（包含价格召回、情感分析和前瞻预测）。
*   **Benchmark 与对比方法**：
    *   **Raw Alpha**：未经过滤的 7 个模型平均信号。
    *   **Debiased Alpha**：简单的阈值过滤法（将高污染信号置零）。
    *   **传统基准**：等权重买入持有（EW Buy-Hold）、20 日动量策略（Momentum-20d）、随机策略（Random）。

*   **实验规模**：涵盖了 7 个不同家族和规模的模型（GPT-2 系列、TinyLlama、Phi-2、Qwen2.5），参数量从 124M 到 7B。

#### 6. 主要结论与发现

*   **收益差异**：清洁信号的平均日收益率为 **14.48 bps**，而受污染信号仅为 **2.13 bps**，证明记忆信号几乎没有真实的预测价值。

#### 7. 优点与亮点
*   **零成本与即插即用**：无需重新训练模型，不改变输入数据（无信息损失），可作为任何 LLM 预测流水线的插件层。
*   **创新的 CMMD 算法**：巧妙利用了模型间训练截止日期的“天然实验”差异，避免了设定硬性阈值的盲目性。
*   **深度洞察**：首次在大规模金融场景下验证了 MIA 的有效性，并揭示了记忆与泛化之间的定量交叉关系。

*   **统计效力受限**：由于仅使用了 93 个交易日进行组合回测，虽然结果显著，但更长周期的验证（如 250 天以上）会使结论更具说服力。
